# Elm learning

## Clone the repo & install:

```
git clone "https://github.com/elm-studies/elm-commands.git"
cd ./elm-commnads

docker compose up -d
docker logs <container name> -f

```

**If necessary, modify the container name and ports defined in the docker-compose.yml file**
